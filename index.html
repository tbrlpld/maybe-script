When the script loads, then this should not show.
When there is a tag with the src, but the source does not exist (404), so this content should show.
When there is no corresponding <script> tag, the content should show.
When the script is not deferred in the head, then it should already be loaded by the time the element is discovered. It should not be waiting for any events in this case. The content should not show.
When the script is not deferred in the head, but there the source if not found (404) then the content should show.
